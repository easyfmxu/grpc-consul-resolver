# gRPC Consul resolver
This lib resolves Consul services by name. 

# Usage

Somewhere in your `init` code: 
```go
resolver.RegisterDefault(time.Second * 5)
```

Getting connection:
```go
conn, err := grpc.DialContext(ctx, "srv://consul/my-awesome-service")
```
